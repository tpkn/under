# Under
Module for underline text


## Usage

```go
under.Lined(text, symbol string, with_text bool)

// Lorem ipsum dolor sit amet
// ++++++++++++++++++++++++++
```


```go
under.LinedSolid(text string, with_text bool)

// Lorem ipsum dolor sit amet
// ──────────────────────────
```


```go
under.LinedDashed(text string, with_text bool)

// Lorem ipsum dolor sit amet
// --------------------------
```


```go
under.LinedDotted(text string, with_text bool)

// Lorem ipsum dolor sit amet
// ..........................
```
